# Tripleten web_project_around_es

Este proyecto ofrece la posibilidad de descubrir diversas imágenes de distintos lugares de Estados Unidos, además de añadir ubicaciones personalizadas.
Incluye un perfil interactivo y configurable, en el que se pueden editar datos como el nombre, la ocupación, las fotografías y el contenido mostrado.
También permite visualizar las imágenes en mayor tamaño y marcar con “me gusta” las tarjetas preferidas.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
